# AI Image Generation Prompt for Game Background

To generate the perfect background for **Orchard Harvest**, copy and paste the following prompt into an AI image generator like Midjourney, DALL-E 3, or Stable Diffusion:

***

**Prompt:**
> A beautiful, serene apple orchard at golden hour, premium 2D vector art style, flat design, soft pastel colors, warm sunlight filtering through the trees, soft diffuse shadows. The scene is cozy, elegant, and organic. The center of the image should be relatively uncluttered with a soft glowing clearing to allow gameplay elements to stand out. Sage green, warm cream, and wood brown color palette. Studio Ghibli inspired background, high quality, 16:9 aspect ratio.

***

### Instructions for adding it to the game:
1. Generate the image using the prompt above.
2. Save the image as exactly `background.jpg`.
3. **Important:** Place the `background.jpg` file inside the `public/` folder of this project (so the path is `public/background.jpg`). 
*(Note: In Vite/React apps, images that are referenced directly by the browser need to live in the `public` folder to be displayed correctly!)*
4. Refresh your game, and the beautiful new background will automatically appear behind the falling fruits!

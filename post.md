🚀 Day 16 of 30 Days of Vibe Coding: Lumina Workspace! ✨

For Day 16, I built the ultimate spatial workspace: a premium, mobile-optimized Sticky Notes app! I wanted to create something that felt incredibly tactile, polished, and useful across all devices.

Meet Lumina Workspace. 📝

Here is what I built for Day 16: 👇

📌 Spatial Canvas & Drag-and-Drop:
You can spawn sticky notes and drag them anywhere on the screen. I used Framer Motion to handle the drag physics, making it feel incredibly smooth and responsive.

📐 Custom Mobile-Optimized Resizing:
Native CSS resizing is notoriously bad on mobile touch screens. So, I built a custom resize handle from scratch using pointer events! Now you can smoothly resize any note on both desktop and mobile devices.

🌓 Dark & Light Mode:
The app defaults to a gorgeous deep OLED dark theme, but features a seamless toggle to a crisp snow-white light theme. The background is a subtle dotted grid that adapts perfectly to both modes.

🎨 Premium UI:
The header and footer are full-width sticky bars with a beautiful frosted glass effect, anchoring your workspace. The notes themselves use premium pastel colors and a handwriting font (`Caveat`) to give them an organic feel.

🛠️ The Tech Stack:
- React 19 & Vite
- Tailwind CSS 4 (with custom CSS variables for theming)
- Framer Motion (for drag controls and animations)
